# PorownanieAlgorytmowZastepowaniaStron
Celem projektu jest porównanie skuteczności dwóch klasycznych algorytmów zastępowania stron w pamięci operacyjnej:
LRU (Least Recently Used) oraz LFU (Least Frequently Used)


Badanie ma na celu sprawdzenie, który z algorytmów lepiej minimalizuje liczbę błędów strony przy różnych rozmiarach pamięci podręcznej (ramkach)
